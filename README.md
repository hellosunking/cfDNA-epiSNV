### Codes and scripts used in Zhang et al.

Please refer to each directory (where `wk.sh` is the main script) for detailed information.

Notes:
1. `wk_wgs.sh` is designed for WGS data process, while `wk_EM-seq.sh` is for EM-seq data.
2. `Ktrim`, `bwa`, `picard`, `Msuite2`, `samtools`, `bedtools`, and `R` are required to run these codes.
